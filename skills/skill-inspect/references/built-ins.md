# Built-in Claude Code Commands

These are hardcoded in the Claude Code application and cannot be found in the filesystem.

| Command | Description |
|---------|-------------|
| `/help` | Show available commands and usage |
| `/clear` | Clear conversation history |
| `/compact` | Compact conversation context |
| `/model` | Switch AI model |
| `/cost` | Show token usage and cost |
| `/login` | Authenticate with Anthropic |
| `/logout` | Sign out of current session |
| `/doctor` | Diagnose Claude Code installation |
| `/permissions` | Manage tool permissions |
| `/mcp` | List and manage MCP servers |
| `/config` | View and edit settings |
| `/memory` | View project memory (CLAUDE.md) |
| `/vim` | Toggle vim keybinding mode |
| `/review` | Review recent changes |
| `/init` | Initialize project configuration |
| `/status` | Show session status |
| `/listen` | Listen mode (paste-friendly) |
| `/ide` | IDE integration controls |
| `/terminal` | Open terminal panel |
| `/bug` | Report a bug |
| `/fast` | Toggle fast mode (same model, faster output) |
